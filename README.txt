== Installation ==

1. Upload the entire `WhatsFab` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **WhatsFab** menu in your WordPress admin screen.
